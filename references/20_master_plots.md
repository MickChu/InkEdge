# 20 Master Plots — 情节基因引擎

> Ronald B. Tobias, *20 Master Plots: And How to Build Them* (1993)
> 核心理念：套路不单独存在，组合产生新故事。20 种基础情节是"基因"，2-3 组混合即可生成独特骨架。

---

## 20 种基础套路

### 1. Quest（追寻）
主角上路找一个具体目标（物品、地点、人）。重点是旅程本身对主角的改变。
- 例：《魔戒》、《西游记》
- 组合：+ Discovery = 考古探险；+ Rivalry = 竞赛寻宝

### 2. Adventure（冒险）
主角被抛入未知世界，核心是探索而非寻找。没有特定目标，一切相遇都是意外。
- 例：《格列佛游记》、《海底两万里》
- 组合：+ Discovery = 新大陆探索；+ Escape = 异世界求生

### 3. Pursuit（追捕）
A 追 B。推进力来自"能否逃脱/抓获"的悬念。双方都在赌。
- 例：《亡命天涯》、《猫鼠游戏》
- 组合：+ Revenge = 血亲复仇追杀；+ Mystery = 追查真凶

### 4. Rescue（拯救）
主角去救被困的人。道德驱动力强，核心是"值得冒险吗"。
- 例：《拯救大兵瑞恩》、《飓风营救》
- 组合：+ Sacrifice = 舍身救人；+ Underdog = 弱者孤身营救

### 5. Escape（逃脱）
主角被困，需要脱身。核心是智斗+意志力，通常反体制。
- 例：《肖申克的救赎》、《大逃亡》
- 组合：+ Rivalry = 越狱对决；+ Transformation = 囚禁→蜕变

### 6. Revenge（复仇）
主角因冤屈而报复。核心是道德模糊——复仇是否让他变成怪物？
- 例：《基督山伯爵》、《杀死比尔》
- 组合：+ Pursuit = 千里追杀；+ Temptation = 复仇途中被诱惑

### 7. The Riddle（谜题）
主角破解一个谜。核心是智力的快感，通常藏在表面之下。
- 例：《达芬奇密码》、《东方快车谋杀案》
- 组合：+ Discovery = 破解历史谜题；+ Pursuit = 限时解谜

### 8. Rivalry（对抗）
两个势均力敌的人/势力对抗。核心不是仇恨而是"必须赢"。
- 例：《莫扎特传》、《爆裂鼓手》
- 组合：+ Underdog = 弱者挑战巨人；+ Love = 情敌

### 9. Underdog（逆袭）
弱者对抗压倒性优势。读者天然站弱者一边。
- 例：《洛奇》、《国王的演讲》
- 组合：+ Revenge = 被欺压后翻盘；+ Adventure = 小人物闯世界

### 10. Temptation（诱惑）
主角被诱惑，做出道德妥协，最终面临后果。
- 例：《浮士德》、《华尔街之狼》
- 组合：+ Descension = 诱惑→堕落；+ Sacrifice = 最后时刻拒绝

### 11. Metamorphosis（蜕变）
主角经历身体的/生理的转变。核心是"变成另一个人意味着什么"。
- 例：《变形记》、《苍蝇》
- 组合：+ Discovery = 变身后发现新能力；+ Escape = 变异中求生

### 12. Transformation（转化）
主角经历心理的/灵魂的转变。与 Metamorphosis 不同：外表不变，内在改变。
- 例：《圣诞颂歌》、《小王子》
- 组合：+ Maturation = 少年→成人；+ Sacrifice = 自私→无私

### 13. Maturation（成长）
主角从天真走向成熟。通常标记"失去纯真"的转折点。
- 例：《麦田里的守望者》、《杀死一只知更鸟》
- 组合：+ Adventure = 冒险中长大；+ Underdog = 弱小孩童→强者

### 14. Love（爱情）
两个人在阻力中相爱。核心不是"在一起"而是"什么在阻止他们"。
- 例：《傲慢与偏见》、《罗密欧与朱丽叶》
- 组合：+ Forbidden Love = 阶级/家族阻隔；+ Rivalry = 情敌三角

### 15. Forbidden Love（禁忌之恋）
爱情本身被社会/法律/道德禁止。比 Love 更具张力。
- 例：《洛丽塔》、《断背山》
- 组合：+ Sacrifice = 为爱抛弃一切；+ Escape = 私奔

### 16. Sacrifice（牺牲）
主角为了更大的善放弃珍贵之物。核心是代价——"值得吗"。
- 例：《双城记》、《美丽人生》
- 组合：+ Rescue = 舍身救人；+ Transformation = 牺牲本身是转化

### 17. Discovery（发现）
主角发现隐藏的真相/知识/文明。核心是"知道了不该知道的事"。
- 例：《侏罗纪公园》、《接触》
- 组合：+ The Riddle = 发现即解谜；+ Adventure = 发现新世界

### 18. Wretched Excess（极端放纵）
主角沉溺于某种极端行为，最终被它摧毁。
- 例：《了不起的盖茨比》、《道林·格雷的画像》
- 组合：+ Temptation = 放纵的起点；+ Descension = 放纵的终点

### 19. Ascension（崛起）
主角从低处爬到高处。核心是权力的代价和孤独。
- 例：《教父》、《社交网络》
- 组合：+ Underdog = 底层崛起；+ Revenge = 爬上去清算旧账

### 20. Descension（堕落）
主角从高处坠落。核心是悲剧感——他知道不该这么做，但停不下来。
- 例：《麦克白》、《绝命毒师》
- 组合：+ Temptation = 被引诱→堕落；+ Ascension = 先崛起后坠落

## 组合逻辑

Tobias 的核心论点：**没有纯套路**。任何故事都是 2-3 个基础基因的混合体。

### 常见组合公式

| 组合 | 产物 | 例子 |
|------|------|------|
| Underdog + Ascension | 底层崛起 | 《洛奇》、《国王的演讲》 |
| Revenge + Descension | 复仇者自毁 | 《基督山伯爵》（原著的复杂版） |
| Discovery + The Riddle | 发现谜题 | 《达芬奇密码》 |
| Escape + Transformation | 囚禁中成长 | 《肖申克的救赎》 |
| Love + Sacrifice | 为爱牺牲 | 《双城记》 |
| Quest + Maturation | 寻路长大 | 《魔戒》（Frodo的弧线） |
| Temptation + Descension | 堕落之路 | 《浮士德》、《绝命毒师》 |
| Underdog + Discovery + Sacrifice | 弱者发现奇迹并牺牲 | 《匣中纪元》 |

### 组合规则
1. **相斥套路不混用**：Quest（向外找）和 Escape（向外逃）可以并存，但 Discovery（发现）和 Metamorphosis（变形）是先后关系不是并列
2. **不超过 3 个**：2 个最优，3 个是黄金上限
3. **一个做主轴，其余做副线**：主轴驱动外部情节，副线驱动角色弧线

## 与 Snowflake Method 的协作

| Snowflake 步骤 | 对应使用 20MP 的方式 |
|---------------|---------------------|
| Step 1: 一句话核心 | 从中识别 1-2 个基础套路作为"故事基因" |
| Step 2: 一段话扩展 | 明确主轴套路和副轴套路 |
| Step 3: 角色概要 | 每个角色的弧线映射到一个套路（如"主角=Underdog→Ascension"） |
| Step 4+: 大纲展开 | 确保每章至少推进主轴套路的一个节拍 |

## InkEdge 集成方式

```
python main.py new --name "书名" --topic "..." --plots Underdog,Discovery,Sacrifice
```

Architect 在生成 foundation 时，将选定的套路注入 prompt，约束以下维度：
- story_frame: 主轴套路定义核心命题
- volume_map: 每卷对应套路节拍的推进
- roles: 每个角色的弧线映射套路
